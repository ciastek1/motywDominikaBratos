Motyw Wordpress Księgarni "Słowo w słowo" | Dominika Bratos

Opis projektu:
Projekt przedstawia motyw, który został przygotowany na podstawie strony onepage Księgarni "Słowo w słowo".
Strona zawiera sekcje: powitalną, opisu firmy, oferty oraz formularz kontaktowy.
Motyw został wykonany za pomocą PHP, CSS, HTML oraz JavaScript.

Podczas wykonywania projektu nauczyłam się:
- tworzenia własnych motywów w WordPress

Największą trudnością było:
- poprawne połączenie plików

Zdjęcia pochodzą ze strony Pexels https://www.pexels.com/
